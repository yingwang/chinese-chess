# Chinese Chess

A Chinese Chess (Xiangqi) game with AI opponent, built with C++ and SDL2.

## Features

- Full Xiangqi rules: General, Advisor, Elephant, Horse, Chariot, Cannon, Soldier
- AI opponent using minimax search with alpha-beta pruning (depth 4)
- Piece-value and position-based board evaluation
- SDL2 rendering with texture-based board and pieces

## Tech Stack

C++, SDL2, Xcode

## How to Build

1. Open `ChineseChess.xcodeproj` in Xcode
2. Ensure SDL2 is installed (e.g., `brew install sdl2 sdl2_image sdl2_ttf`)
3. Build and run
